1\. 掌握 Git 的基本操作（clone、add、commit、push、pull、branch）

2\. 理解 Pull Request 的完整流程

3\. 学会阅读和理解开源项目的文档

4\. 期末前能提交一次真正的 Pull Request（不管大小）
